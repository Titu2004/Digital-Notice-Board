# Digital Notice Board System

A web-based Digital Notice Board developed using Flask and SQLite.
It allows administrators to post, edit, and delete notices which are displayed in real-time.

## Features
- Admin Login
- Add / Edit / Delete Notices
- Auto Expiry of Notices
- Auto Refresh Display
- Secure Logout

## Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS

## How to Run
1. Install Flask
   pip install flask
2. Run application
   python app.py
3. Open browser
   http://127.0.0.1:5000/
